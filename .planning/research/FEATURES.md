# Feature Landscape

**Domain:** OpenRouter 기반 AI 채팅 앱 (ChatGPT 스타일)
**조사일:** 2026-03-30
**모드:** Ecosystem (기능 분류)

## Table Stakes (필수 기능)

사용자가 기대하는 기본 기능. 없으면 제품이 불완전하게 느껴짐.

| 기능 | 기대 이유 | 복잡도 | 비고 |
|------|----------|--------|------|
| **실시간 SSE 스트리밍** | ChatGPT 표준 경험, 토큰 단위 응답 | 중간 | `text/event-stream`, `data: [DONE]` 처리 필요 |
| **마크다운 렌더링** | 코드블록, 리스트, 헤더 등 서식 | 중간 | react-markdown + remark-gfm + rehype-highlight |
| **점진적 마크다운 렌더링** | 스트리밍 중 부분 마크다운 표시 | 높음 | 불완전한 마크다운 파싱 처리, 코드블록 닫힘 처리 |
| **코드블록 구문 강조** | 코드 가독성 필수 | 낮음 | rehype-highlight로 180+ 언어 지원 |
| **코드블록 복사 버튼** | 사용자 편의성 | 낮음 | 클립보드 API, 토스트 피드백 |
| **대화 목록 (사이드바)** | 대화 전환, 이력 관리 | 중간 | IndexedDB 저장, 최신순 정렬 |
| **새 대화 생성** | 세션 시작 | 낮음 | UUID 생성, 빈 메시지 배열 |
| **대화 삭제** | 관리 용이성 | 낮음 | Cascade 삭제 (대화+메시지) |
| **대화 제목 자동 생성** | 대화 식별 | 중간 | 첫 사용자 메시지 기반, 요약 모델 호출 |
| **메시지 전송/수신** | 채팅 핵심 기능 | 중간 | 사용자/어시스턴트 role 구분 |
| **Stop 버튼** | 스트리밍 중단 제어 | 중간 | AbortController, 패칭 취소 |
| **모델 선택** | 모델 전환 기능 | 중간 | OpenRouter 모델 목록 조회, 무료 필터링 |
| **기본 모델 설정** | 편의성 | 낮음 | 설정 저장, 자동 선택 |
| **시스템 프롬프트 설정** | AI 동작 제어 | 중간 | 대화별/글로벌 설정, UI 분리 |
| **API 키 관리** | OpenRouter 인증 | 중간 | 등록/변경/删除, localStorage |
| **다크/라이트 테마** | 사용자 기대 | 낮음 | Tailwind dark mode, 시스템 설정 감지 |
| **반응형 레이아웃** | 모바일 지원 | 중간 | 사이드바 토글, Sheet/Drawer, 1024px 브레이크포인트 |
| **오토스크롤** | 스트리밍 추적 | 낮음 | `scrollTop = scrollHeight`, 사용자 스크롤 중 감지 |
| **로딩 인디케이터** | 응답 대기 피드백 | 낮음 | 타이핑 애니메이션 (3점) |
| **입력 영역 자동 높이** | 장문 입력 지원 | 낮음 | `autoHeight` textarea |
| **Enter 전송, Shift+Enter 줄바꿈** | 표준 채팅 동작 | 낮음 | 키보드 이벤트 처리 |
| **에러 처리** | 실패 시 안내 | 중간 | API 에러, 네트워크 에러, 재시도 버튼 |

## Differentiators (차별화 기능)

경쟁 제품과 차별화되는 기능. 필수는 아니지만 가치 있음.

| 기능 | 가치 제안 | 복잡도 | 비고 |
|------|----------|--------|------|
| **로컬 전용 저장** | 프라이버시, 오프라인 가능 | 낮음 | IndexedDB, 서버 없음 |
| **무료 모델만 사용** | 비용 제로, 접근성 | 낮음 | OpenRouter 무료 필터링 |
| **대화 제목 자동 생성 (요약)** | UI 편의성 | 중간 | 별도 모델 호출, 타이밍 최적화 |
| **점진적 마크다운 렌더링 (고도화)** | 스트리밍 시각적 피드백 | 높음 | 불완전한 코드블록 처리, 플레이스홀더 |
| **스트리밍 속도 제어** | 가독성 조절 | 중간 | 딜레이 옵션, 속도 슬라이더 |
| **메시지 재전송 (Regenerate)** | 응답 다시 받기 | 낮음 | 마지막 어시스턴트 메시지 삭제 후 재요청 |
| **메시지 복사** | 사용자 편의성 | 낮음 | 전체 메시지 텍스트 복사 |
| **키보드 단축키** | 파워 유저 지원 | 중간 | `Cmd+K` (새 대화), `Cmd+/` (설정) 등 |
| **코드 전체 복사 버튼** | 코드 블록 편의성 | 낮음 | 코드블록 헤더에 복사 아이콘 |
| **대화 검색** | 대화 빠른 찾기 | 중간 | Dexie.js 쿼리, 텍스트 매칭 |
| **입력 힌트/플레이스홀더** | 온보딩 개선 | 낮음 | "메시지를 입력하세요..." |
| **빈 상태 안내** | 첫 사용자 가이드 | 낮음 | "새 대화를 시작하세요" |
| **토큰 사용량 표시** | 투명성 | 낮음 | 추정 토큰 수 (입력/출력) |
| **응답 시간 표시** | 성능 가시성 | 낮음 | 지연 시간 (ms) |

## Anti-Features (안티 기능)

의도적으로 구현하지 않을 기능.

| 안티 기능 | 회피 이유 | 대안 |
|----------|----------|------|
| **PWA/오프라인 지원** | 로컬 환경에서 항상 온라인 상태 | 필요 없음 |
| **대화 내보내기/가져오기 (JSON)** | v2에서 고려, 초기 범위 초과 | IndexedDB 백업으로 대체 |
| **유료 모델 지원** | 무료 모델만 사용 목적, 복잡도 증가 | 무료 모델 필터링 유지 |
| **토큰 사용량 추적/비용 계산** | 무료 모델만 사용 | 필요 없음 |
| **이미지/파일 첨부 (멀티모달)** | 복잡도 높음, v2에서 고려 | 텍스트 전용 유지 |
| **다국어 지원 (i18n)** | 한국어 UI만 필요 | 하드코딩 한국어 |
| **사용자 인증/로그인** | 로컬 전용, 백엔드 없음 | 필요 없음 |
| **클라우드 동기화** | 로컬 전용, 프라이버시 | 필요 없음 |
| **대화 분기 (Forking)** | 복잡도 높음, 사용 빈도 낮음 | 단순 대화 삭제 후 재생성 |
| **대화 공유/링크** | 로컬 전용 | 필요 없음 |
| **대화 목록 날짜별 그룹화** | 단순 최신순 목록으로 충분 | 최신순 정렬 유지 |
| **배포 (Vercel, Netlify 등)** | 로컬 개발 환경 전용 | 로컬 실행만 지원 |
| **음성 입력/출력** | 복잡도 높음, v2에서 고려 | 텍스트 입력 유지 |
| **웹 브라우징 통합** | 복잡도 높음, 외부 API 필요 | 텍스트 생성만 |
| **플러그인/확장 시스템** | 복잡도 높음, 초기 범위 초과 | 필요 없음 |
| **사용자 분석/로그** | 프라이버시 우선 | 필요 없음 |

## Feature Dependencies

```
API 키 관리 → 모델 목록 조회 → 모델 선택
새 대화 생성 → 메시지 전송 → SSE 스트리밍 → 마크다운 렌더링
메시지 전송 → Stop 버튼 (스트리밍 중)
메시지 전송 → 대화 제목 자동 생성 (첫 메시지)
시스템 프롬프트 설정 → 메시지 전송 (프롬프트 포함)
반응형 레이아웃 → 사이드바 토글 (모바일)
마크다운 렌더링 → 코드블록 구문 강조
마크다운 렌더링 → 코드블록 복사 버튼
```

## MVP Recommendation

**Phase 1 (초기 릴리스):**
1. API 키 관리 (등록/변경/삭제)
2. 새 대화 생성
3. 메시지 전송/수신 (SSE 스트리밍)
4. 기본 마크다운 렌더링
5. 코드블록 구문 강조
6. Stop 버튼
7. 대화 목록 (사이드바)
8. 대화 삭제
9. 모델 선택 (무료 모델 필터링)
10. 다크/라이트 테마
11. 반응형 레이아웃 (모바일 사이드바)
12. 오토스크롤
13. 입력 영역 자동 높이
14. Enter 전송, Shift+Enter 줄바꿈
15. 로딩 인디케이터

**Phase 2 (안정화):**
1. 대화 제목 자동 생성
2. 점진적 마크다운 렌더링
3. 코드블록 복사 버튼
4. 기본 모델 설정
5. 시스템 프롬프트 설정
6. 에러 처리 (재시도 버튼)
7. 메시지 재전송 (Regenerate)
8. 메시지 복사
9. 키보드 단축키

**연기 (v2 고려):**
- 대화 검색: Phase 2 이후, 사용 빈도 확인 후 구현
- 토큰 사용량 표시: 유료 모델 도입 시 고려
- 음성 입력/출력: v2에서 기술적 가능성 검토
- 이미지/파일 첨부: v2에서 멀티모달 지원 검토

## UI/UX 특이사항

### SSE 스트리밍 렌더링
- **타이핑 효과:** 토큰 도착 시마다 실시간 추가 (부드러운 애니메이션)
- **점진적 마크다운:** 불완전한 마크다운 파싱 처리
  - 열린 코드블록 플레이스홀더: ```lang 표시 후 닫힘 대기
  - 불완전한 리스트, 헤더: 파서가 tolerant하게 처리
- **오토스크롤:** 새 토큰 도착 시마다 하단 스크롤 (사용자가 수동 스크롤 중이면 자동 스크롤 중지)

### 마크다운 코드블록
- **구문 강조:** 180+ 언어 지원 (rehype-highlight)
- **복사 버튼:** 코드블록 우상단에 아이콘, 클릭 시 전체 코드 복사
- **언어 라벨:** 코드블록 상단에 언어 표시 (예: `javascript`, `python`)

### 반응형 레이아웃
- **데스크톱 (>1024px):** 사이드바 고정 (280px)
- **모바일 (<1024px):** 사이드바를 Sheet/Drawer로 토글
- **사이드바:** 새 대화 버튼 + 대화 목록 + 설정 링크

### 다크모드
- **기본 활성:** 시스템 설정 감지, 기본 다크모드
- **테마 전환:** 설정에서 다크/라이트 수동 전환 가능

### 대화 제목 자동 생성
- **타이밍:** 첫 사용자 메시지 전송 후 비동기 생성
- **방법:** 별도 짧은 모델 호출으로 제목 요약
- **Fallback:** 생성 실패 시 첫 메시지의 앞 30자 사용

## 복잡도 분석

### 낮음 (Low)
- 설정: API 키, 기본 모델, 시스템 프롬프트, 테마
- UI 컴포넌트: 코드블록 복사 버튼, 입력 영역 자동 높이, 로딩 인디케이터
- 기본 동작: Enter 전송, 오토스크롤, 메시지 복사

### 중간 (Medium)
- SSE 스트리밍 구현: fetch API, AbortController, 파싱
- 마크다운 렌더링: react-markdown, 플러그인 설정
- 대화 관리: IndexedDB CRUD, 캐싱
- 모델 목록 조회: OpenRouter API 호출, 필터링
- 반응형 레이아웃: 사이드바 토글, 브레이크포인트
- 대화 제목 자동 생성: 비동기 요약, 타이밍 처리

### 높음 (High)
- 점진적 마크다운 렌더링: 불완전한 마크다운 파싱, 코드블록 플레이스홀더
- 에러 처리: 다양한 에러 케이스, 재시도 로직, 사용자 피드백
