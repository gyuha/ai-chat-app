# Milestone v0.1-alpha: Initial Foundation (Phase 2-3)

**Status:** ✅ SHIPPED 2026-03-31
**Phases:** 2-3
**Total Plans:** 6

## Overview

OpenRouter Chat의 IndexedDB 영속성, ChatGPT 스타일 레이아웃, 대화 관리 기능を含む最初の実演版。Phase 1 (Foundation)은 별도 브랜치에서 작업됨.

## Phases

### Phase 2: Persistence + Layout

**Goal**: IndexedDB에 대화와 설정이 영속적으로 저장되고, ChatGPT 스타일 레이아웃이 기본 형태 갖추어짐
**Depends on**: Phase 1
**Plans**: 3 plans

Plans:

- [x] 02-01: Dexie liveQuery reactive hooks and Zustand UI store for sidebar/conversation state
- [x] 02-02: ChatGPT-style responsive sidebar (260px fixed) with overlay mobile toggle
- [x] 02-03: HomePage integrated with ChatLayout sidebar and IndexedDB persistence

**Details:**
- IndexedDB 영속성 (DATA-01, DATA-02)
- ChatGPT 스타일 좌측 사이드바 + 우측 채팅 영역 (UI-01)
- 반응형 디자인 (UI-02)

### Phase 3: Conversation Management

**Goal**: 사용자가 대화 목록을 관리하고, 테마를 전환하며, 빈 상태에서 안내를 받을 수 있다
**Depends on**: Phase 2
**Plans**: 3 plans

Plans:

- [x] 03-01: Zustand persist theme store with useTheme hook, Tailwind dark class application, and system theme detection
- [x] 03-02: 사이드바 대화 관리 기능 구현 (새 대화, 정렬, 편집, 삭제, 테마 토글)
- [x] 03-03: Empty state UI with MessageSquare icon card and auto-generated conversation titles

**Details:**
- 대화 관리 (CONV-01 ~ CONV-06)
- 테마 시스템 (UI-03)
- 빈 상태 안내 (UI-04)
