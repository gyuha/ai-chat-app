**app.css dark mode 색상 검증:**
UI-SPEC 색상표와 일치하는지 확인:
```css
.dark {
  --background: 0 0% 7%;          /* #0D1117 */
  --foreground: 0 0% 98%;
  --secondary: 0 0% 14.9%;        /* #161B22 */
  --primary: 199 89% 48%;         /* #19A3FD */
  --accent: 199 89% 48%;          /* #19A3FD */
  /* ... */
}
```

app.css의 dark mode 색상이 UI-SPEC 색상(#0D1117, #161B22, #19A3FD)과 일치하는지 확인하고 불일치 시 수정한다.
