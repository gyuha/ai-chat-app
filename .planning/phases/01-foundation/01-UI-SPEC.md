### Accent Reserved For
- Primary CTA buttons (e.g., "API 키 저장", "전송")
- Send button icon
- Active model selector indicator
- Loading spinner
- Links in text

**NOT for:** Border colors, passive icons, secondary actions

## Component Inventory

### API Key Input (AUTH-01, AUTH-02)
- **Component:** Input field + Primary button
- **States:** Empty, Filled, Validating (loading), Valid (green check), Invalid (red check + error message)
- **Behavior:** Debounced validation on blur (GET /api/v1/models)
- **Copy:** "OpenRouter API 키 입력" (placeholder), "저장" (button)

### Model Selector (MODL-03)
- **Component:** Dropdown (shadcn Select)
- **States:** Loading (skeleton), Loaded, Selected, Error
- **Behavior:** Filter free models (pricing.prompt === "0" && pricing.completion === "0"), cached 10min via TanStack Query
- **Copy:** "모델 선택" (label), model name as option

### Chat Input (CHAT-01)
- **Component:** Textarea + Send button
- **States:** Empty (disabled send), Has text (enabled send), Sending (loading)
- **Behavior:** Enter to send (Shift+Enter for newline), disabled during streaming
- **Copy:** "메시지를 입력하세요..." (placeholder)

### Chat Message (CHAT-01, CHAT-02, CHAT-03)
- **Component:** Message bubble (user right-aligned, AI left-aligned)
- **User bubble:** Accent background, white text
- **AI bubble:** Secondary background, foreground text, markdown rendered
- **States:** Sending (streaming animation), Complete, Error
- **Streaming behavior:** Token-by-token update, Markdown render on complete OR when code block starts

### Stop Button (CHAT-05)
- **Component:** Destructive secondary button
- **States:** Hidden (not streaming), Visible (streaming)
- **Copy:** "중단"
- **Behavior:** AbortController cancellation

### Loading Indicator (CHAT-04)
- **Component:** Typing indicator dots or spinner
- **Style:** 3 dots pulsing animation, accent color
- **Position:** AI message area, replacing empty state

### Toast Notifications (D-09)
- **Component:** shadcn Toast
- **Types:** Success (green), Error (red), Info (accent)
- **Behavior:** Auto-dismiss 5s, manual dismiss available
- **Copy examples:**
  - Error: "API 키가 유효하지 않습니다. 다시 확인해 주세요."
  - Rate limit: "요청이 너무 많습니다. 잠시 후 다시 시도해 주세요."

### Settings Page (AUTH-03)
- **Route:** /settings
- **Components:** API Key input (masked), Model selector, System prompt textarea
- **Copy:** "설정", "API 키", "기본 모델", "시스템 프롬프트"

### Empty State (Phase 1 focus: API Key not set)
- **Component:** Centered card with icon + text + CTA
- **Copy:** "OpenRouter API 키가 필요합니다" (heading), "무료 AI 모델과 대화하려면 API 키를 입력해 주세요." (body), "API 키 입력" (CTA)
