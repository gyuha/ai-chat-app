Zustand persist middleware를 사용하여 theme preference를 localStorage에 저장하는 store를 생성한다.

파일 생성: `src/stores/theme-store.ts`

```typescript
import { create } from "zustand"
import { persist } from "zustand/middleware"

export type Theme = "light" | "dark" | "system"

interface ThemeState {
  theme: Theme
  setTheme: (theme: Theme) => void
}

export const useThemeStore = create<ThemeState>()(
  persist(
    (set) => ({
      theme: "system",
      setTheme: (theme) => set({ theme }),
    }),
    {
      name: "theme-preference",
    }
  )
)
```

**Key points:**
- Theme type: "light" | "dark" | "system"
- Default: "system"
- localStorage key: "theme-preference"
- persist middleware handles localStorage automatically

Theme store의 값을 구독하고, <html> element에 Tailwind dark class(.dark)를 적용하는 hook을 생성한다.

파일 생성: `src/hooks/use-theme.ts`

```typescript
import { useEffect } from "react"
import { useThemeStore } from "@/stores/theme-store"

export function useTheme() {
  const theme = useThemeStore((s) => s.theme)

  useEffect(() => {
    const root = window.document.documentElement

    root.classList.remove("light", "dark")

    if (theme === "system") {
      const systemTheme = window.matchMedia("(prefers-color-scheme: dark)").matches
        ? "dark"
        : "light"
      root.classList.add(systemTheme)
    } else {
      root.classList.add(theme)
    }
  }, [theme])

  // Listen for system theme changes when in "system" mode
  useEffect(() => {
    if (theme !== "system") return
    const mediaQuery = window.matchMedia("(prefers-color-scheme: dark)")
    const handler = () => {
      const root = window.document.documentElement
      root.classList.remove("light", "dark")
      root.classList.add(mediaQuery.matches ? "dark" : "light")
    }
    mediaQuery.addEventListener("change", handler)
    return () => mediaQuery.removeEventListener("change", handler)
  }, [theme])
}
```

**Theme application rules:**
- Always remove both classes first
- If theme="system": detect via matchMedia, add "dark" or "light"
- If theme="light" or "dark": add that class directly
- Listen for system theme changes only when in system mode
