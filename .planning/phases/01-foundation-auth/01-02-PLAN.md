<interfaces>
HTTP contract to implement exactly:

```http
POST /auth/signup
Body: { "email": string, "password": string }
Response 201: { "user": { "id": string, "email": string } }
```

```http
POST /auth/login
Body: { "email": string, "password": string }
Response 200: { "user": { "id": string, "email": string } }
Set-Cookie: access_token=...; HttpOnly
Set-Cookie: refresh_token=...; HttpOnly
```

```http
GET /auth/session
Response 200: { "user": { "id": string, "email": string } }
Response 401: { "message": "Unauthorized" | similar Nest default }
```

```http
POST /auth/logout
Response 200: { "ok": true }
Clears cookies: access_token, refresh_token
```
