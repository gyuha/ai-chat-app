1. `useSetting(key: string)` - TanStack Query `useQuery` 사용:
   - queryKey: `['settings', key]`
   - queryFn: `const setting = await db.settings.get(key); return setting?.value ?? null;`
   - 반환 타입: `UseQueryResult<string | null>`

2. `useSaveSetting()` - TanStack Query `useMutation` 사용:
   - mutationFn: `async ({ key, value }: { key: string; value: string }) => { await db.settings.put({ key, value }); }`
   - onSuccess: `queryClient.invalidateQueries({ queryKey: ['settings', key] })`
   - 반환 타입: `UseMutationResult<void, Error, { key: string; value: string }>`

3. `useDeleteSetting()` - TanStack Query `useMutation` 사용:
   - mutationFn: `async (key: string) => { await db.settings.delete(key); }`
   - onSuccess: `queryClient.invalidateQueries({ queryKey: ['settings', key] })`

설정 key 상수를 export:
```typescript
export const SETTINGS_KEYS = {
  API_KEY: 'apiKey',
  DEFAULT_MODEL_ID: 'defaultModelId',
  SYSTEM_PROMPT: 'systemPrompt',
} as const;
```
