
#  프롬프트

"OpenRouter의 무료 api를 이용한 웹 채팅 어플리케이션을 제작 하려고 함. api의 키를 감추기 위해서 웹 앱과 서버를 만들어서 사용 하려고 함. backend는 Nest.Js, typescript로 제작하고, frontend는 react, typescript, shadcn ui, zustand, tanstack-query, tanstack-router, pnpm, biome 등을 사용 해 줘."

get-shit-done(https://github.com/gsd-build/get-shit-done)과 (https://github.com/nextlevelbuilder/ui-ux-pro-max-skill)을 같이 써서 사이트 개발을 하는 워크 플로우 및 필요한 프롬프트를 작성 해 줘
