
#  프롬프트

```
OpenRouter의 무료 api를 이용한 웹 채팅 어플리케이션을 제작 하려고 함. 
UI는 챕GPT와 유사한 모양으로 만들어 줘.

프론트만 있는 프로젝트 이고 기술 스택은 아래와 같음
    - react
    - pnpm
    - biome
    - typescript
    - shadcn ui

    - tanstack-query
    - tanstack-router
```
